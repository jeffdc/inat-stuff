Troubleshooting CORS Issues

If you get a CORS/NetworkError, try these solutions:

  1. Enable CORS proxy: Visit cors-anywhere.herokuapp.com/corsdemo and request temporary access
  2. Use a local web server: Instead of opening the HTML file directly, serve it via:
    • Python: python -m http.server 8000 then visit http://localhost:8000
    • Node.js: npx http-server
    • VS Code: Use "Live Server" extension
  3. Browser extension: Install a CORS-disabling extension (for development only)
  4. Alternative: Export your iNaturalist data from your profile and load it manually
